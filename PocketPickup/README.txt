Replace the AndroidManifest.xml, as well as the 'res' and 'src' directories in your eclipse pocketpickup workspace with the ones provided in this repo. Update the meta tag value that says "YOUR_KEY_GOES_HERE" in AndroidManifest.xml with your Google Maps API v2 API key. Also update where it says "YOUR_APPLICATION_ID_GOES_HERE" and "YOUR_CLIENT_KEY_GOES_HERE" in src/com/uwcse403/pocketpickup/PocketPickupApplication.java with an application id and a client key.
